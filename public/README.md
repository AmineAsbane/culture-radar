# Paris Culture Anonyme - Site Vitrine

Site vitrine responsive pour la découverte d'événements culturels à Paris, conçu avec HTML5, CSS3 et JavaScript Vanilla.

## Fonctionnalités

- Design moderne et responsive
- Navigation fluide avec menu mobile
- Section événements interactive
- Carte Google Maps intégrée
- Formulaire de contact fonctionnel
- Animations au défilement
- Compatible avec tous les appareils

## Structure du projet

```
paris-culture-anonyme/
├── index.html          # Page principale
├── css/
│   └── style.css      # Styles CSS
├── js/
│   └── script.js      # Fonctionnalités JavaScript
└── README.md          # Documentation
```

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript Vanilla
- Google Fonts (Montserrat)
- Font Awesome (icônes)
- Google Maps API (intégration de carte)

## Comment lancer le projet

1. Téléchargez ou clonez ce dépôt
2. Ouvrez le fichier `index.html` dans votre navigateur web

## Personnalisation

### Couleurs
Les couleurs principales peuvent être modifiées dans le fichier `css/style.css` en modifiant les variables CSS dans la section `:root`.

### Images
Les images sont chargées depuis Unsplash. Vous pouvez les remplacer en modifiant les URLs dans le HTML.
